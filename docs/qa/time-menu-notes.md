# Time メニュー確認メモ（Block 3）

## 仕様

- `🕒 Time` メニューは Edit モードで利用する。
- 挿入: `(今日)` `(今)` `(日時)`
- 展開: `YYYY-MM-DD` / `HH:mm` / `YYYY-MM-DD HH:mm`
- 展開対象は以下のみ:
  - カーソル直前の単一トークン
  - 選択範囲内の対象トークン

## 手動チェック

1. Edit Panel を `Edit` に切り替えて `🕒 Time` が表示される。
2. 空カーソル位置で `挿入 > (日時)` を押すと `(日時)` が挿入される。
3. `(今日)` の直後にカーソルを置いて `展開 > 日付` で `YYYY-MM-DD` に置換される。
4. 選択範囲に `(今)` を含めて `展開 > 時刻` で `HH:mm` に置換される。
5. 対象外（トークンなし）では toast が表示され、本文は変更されない。
