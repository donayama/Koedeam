# チャンクモデル確認メモ（Block 4）

## 仕様

- 音声 `final` 1回を 1チャンクとして扱う。
- チャンク境界は `――` 区切りで可視化される。
- 最小操作:
  - チャンク削除
  - チャンク分割（カーソル位置）
  - チャンク結合（現在 + 前チャンク）
  - チャンク整形（余分な空白の正規化）

## 手動チェック

1. 音声確定を2回行うと `――` 区切りで2チャンク表示される。
2. 2個目チャンクで「チャンク削除」を押すと1個目のみ残る。
3. 単一チャンク中間で「チャンク分割」を押すと境界が追加される。
4. 2個目チャンクで「チャンク結合」を押すと境界が消える。
5. 「チャンク整形」で連続空白が単一空白に正規化される。
