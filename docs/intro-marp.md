# Koedeam

音声認識で文字入力した結果を編集し、共有するためのツール

出てきたアイディアを即記録。適宜編集し、次に繋げます。

# 音声入力

音声入力する際に認識した文言の追記箇所を選べます。
- 文章の末尾(推奨)
- カーソルの位置


<!--
発表者ノート：
音声入力にあたっては文字入力箇所を文章の一番最後に追記するか、
カーソル位置に挿入するかを選択できますが、
Koedeamでは文章の末尾に追記することを推奨しています。
-->

----

# キーボード＋音声<br>同時編集

音声入力した内容を
キーボードで編集している途中でも
さらに音声入力すすれば追記可能。

（※）「カーソル位置への挿入」では不可

これにより発想を止めることなく、
メモし続けることが可能です。

<!--
発表者ノート：
文章の末尾への追記が推奨なのはなぜか。

このようにすでにある音声入力後のテキストを編集しながらでも、
思いついたことがあれば音声入力をし続けることが可能だからです。

これにより、アイディアをより深めることができ、
内容の精度をどんどん高めていくことができると考えています。

これが、Koedeamの強みだと考えています。

-->

----

# 編集機能

スマホがちょっと苦手？な
カーソル移動、範囲選択もKoedeamなら。



<!--
発表者ノート：
Koedeamでは、PCでは簡単ですが、スマホでは操作しづらい、
編集カーソルや範囲選択を簡単にできるようにする機能を用意しています。

カーソル移動では上下左右移動だけではなく、
文頭・文末・行頭・行末の移動ができます。

範囲選択では、行選択・段落選択などをしたうえで、
範囲を上下に拡大縮小しやすくなっています。
文章の前後入れ替えをするためのカットアンドペーストボタンも用意しています。

ほかにも編集関係のツールバーも用意しています。
-->

----

# 共有機能

全文もしくは選択範囲の内容を
他のアプリ、クリップボード、
SNSや生成AIなどに連携

プリセットの連携先は10個、
追加削除も設定から可能！

<!--
発表者ノート：
入力した内容は別のアプリケーションや
クリップボード、SNS・生成AIに連携できます。

設定先はユーザ側で自由に差し替えられます。

-->

----

# テンプレート機能

生成AIのプロンプトの定型文などがあれば、
タップひとつで追記！
（カーソル位置、文章の最初、最後）


<!--
発表者ノート：
テンプレート機能により事前に用意したテキストを
カーソル位置、文章の最初や最後に追記することができます。

生成AIへ連携するための指示文、
たとえば文章の要約や整理を促すには、
文章の最初にこんな感じで埋め込むことで実現できます。
-->
